Postgraduate Lifecycle Management System

This GitHub Pages site exists only to satisfy the repository Pages publishing source at /docs.

The actual application in this repository is a full Next.js platform with server routes, Firebase Authentication, Supabase Postgres database access, and Supabase Storage integrations. It is not deployable to GitHub Pages as a working app.

To run the application, use a Node-capable deployment target such as Vercel, Azure App Service, Azure Container Apps, or another server environment that supports Next.js.

A repository-grounded workflow report is available at WORKFLOW_REPORT.md.
